## 如何編譯
### Linux
```bash
  $flutter build linux
```
### Android
```bash
  $flutter build apk
```
### Web
- 編譯成web容器: `$make`
- Debug階段(使用本地的chrome輸出編譯結果): `$flutter run -d chrome`
